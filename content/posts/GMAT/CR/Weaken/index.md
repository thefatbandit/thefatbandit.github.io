Weaken type question require you to select the answer choice that undermines the author’s argument as decisively as possible.

{{< alert type="warning" >}}
💡 **MOST FREQUENT CR QUESTION TYPE**
{{< /alert >}}

## Rules to Solve

1. The ***stimulus will always contain an argument*** since you are asked to weaken the author’s reasoning.
2. ***Focus on the conclusion***. The more we know about its specifics, the better armed we will be to deciding the correct answer choice.
3. ***Information in the stimulus is suspect***. Read the arguments carefully
4. 🚨***Pre-phase possible answers***🚨 before proceeding to the answer choices
5. 💡***Answer choices are accepted as given*** even if they include “new information”.

## Identifying Question Type

1. The stem uses words synonymous to “weaken”.
- ***“Weaken” synonyms:*** Weaken, Attack, Undermine, Refute, Call into question, cast doubt, challenge, damage, counter
    
2. The stem usually indicates that you should accept the answer choices as true

***<u>Examples</u>***
- “Which of the following if true, most seriously weakens the argument above?”
- “Which of the following, if true, casts the most doubt on the conclusion drawn above?”
- “Which of the following, if true, most calls into question the claim above?”
- Which of the following, if true, is most damaging to the conclusion above?
- “Which of the following, if known, is evidence that contradicts the hypothesis above?”
- “Which of the following, if discovered, would be evidence against the speculation above?”


## How to weaken an argument

The key is to weaken not attack the conclusion

{{< alert type="danger" >}}
🚨 Attack is not the same as destroy (***Attack ≠ Destroy***). *The answer choices most likely hurt the argument.* 

When evaluating an answer ask ***“Would this answer choice make the author reconsider his/her position or force the author to respond?”***

{{< /alert >}}

### The Premises

Attacking the premises on which the conclusion rests. Rare occurrence since it makes easy spotting of answer choice.

{{< alert type="warning" >}}
💡 One of the ***few rare times a premise is attacked***, it is a ***sub-conclusion*** that is being used to drive in the main conclusion
{{< /alert >}}

## 🚨The Conclusion🚨

Most likely. But the correct answer choice doesn’t simply contradict the conclusion. But rather, it often does so by showing that the conclusion fails to account for some element or possibility.

# How to attack a causal conclusion

There are a few ways you can prepare to weaken/strengthen a causal relationship in GMAT: 

1. ***<u>Cause_2 → Effect</u>*** Find an alternate cause for the stated effect
2. ***<u>Cause → ~~Effect~~</u>*** Show that even when the cause occurs, the effect does not occur
3. ***<u>~~Cause~~ → Effect</u>*** Show that though the effect occurred, the cause did not
4. ***<u>Effect → Cause</u>*** Show that the stated relationship is reversed
5. ***<u>Stat Problem:</u>*** Show that a statistical problem exists with the data used to make the causal statement

## Common Weakening Scenarios

The most common logical fallacies that are to be attacked in GMAT Weaken question stimuli are:

1. **Incomplete Information:** When <u>*author fails to consider all possibilities*</u> or relies upon incomplete evidence. This flaw is <u>*attacked by bringing up new possibilities or information*</u>. 
2. **Improper Comparison:** Author attempts to <u>*compare items that are essentially different*</u>
3. **Overly Broad Comparison:** The author draws a conclusion that is broader than the premises support.

## Answer Traps

1. **<u>Opposite Answers:</u>** These  answer choices are presenting information that relates perfectly to the arguments, but instead of weakening it, they are strengthening it.
2. **<u>Shell Game Answers:</u>** The idea/concept raised is similar to the ones present in the stimulus but changed enough to be incorrect yet attractive.
3. **<u>Out of Scope Answers:</u>** They simply miss the point of the argument & raise unrelated/tangential issues to the argument
