## Lesson Content
The stimuli of Boldface questions can be hodgepodges of background information, premises, conclusions, and information of other types, and often it’s not obvious what roles the various parts of the passages play.

So, what’s the key to determining what roles the statements in a Boldface stimulus play? The key is finding any conclusions that appear in the passage, and in particular, the main conclusion.

The roles of all the statements in an argument tend to depend on how the statements are related to the conclusion of the argument.

Generally there is one main conclusion in a Boldface stimulus, and there are sometimes intermediate conclusions as well.

{{< alert type="warning" >}}
💡<b>MUST KNOW:</b>

A key move in the process of answering a Boldface question is locating the main conclusion of the argument. By finding the main conclusion in the passage of a Boldface question, you set yourself up with a reference point for determining what roles the other parts play.
{{< /alert >}}

Consider the following example:

> <em>Ever since certain password recommendations were released in 2003, many people have decided that<strong> they should use passwords that are based on single words, except that digits and symbols have been substituted for some of the words’ letters</strong>. However, people would be better off using passwords that are simply long strings of random words. Not only can password cracking software be set up to look for symbols typically substituted for letters of words but also, <strong>the longer the password the more difficult it is to crack</strong>, and long strings of random words are easier to memorize than long strings made up of letters, digits, and symbols.</em>

> <em>In the argument given, the two portions in boldface play which of the following roles?</em>

> <em>(A) The first outlines a recommendation made by the author; the second is information that the author feels should be considered in following that recommendation.</em>

> <em>(B) The first is a practice that the author proceeds to recommend against; the second is evidence that the author uses in the process of supporting that recommendation.</em>

If the passage above were to appear in a Boldface question, the roles played by each of the bolded sections might not be immediately clear.

The main conclusion of the passage is not the clause found at the end of the passage, but is, rather, the following:

> <em>people would be better off using passwords that are simply long strings of random words.</em>

We can tell that the above is the main conclusion by noticing the following.

The entire first sentence, <em>“Ever since …</em> <strong><em>substituted for some of the words’ letters</em></strong><em>,” </em>neither serves as support for any conclusion nor is supported by any other statement made.

The entire last sentence, <em>“Not only can password cracking software … long strings made up of letters, digits, and symbols</em>,” serves as support for the second sentence, <em>“However, people … long strings of random words.”</em> So, the second sentence states the main conclusion of the passage.

To confirm that the second and third sentences work together in this way, let’s put them together and add the word “Therefore” before the conclusion:

> <em>Not only can password cracking software be set up to look for symbols typically substituted for letters of words but also, <strong>the longer the password the more difficult it is to crack</strong>, and long strings of random words are easier to memorize than long strings made up of letters, digits, and symbols. Therefore, people would be better off using passwords that are simply long strings of random words.</em>

Now that we clearly see what the main conclusion is, we can readily see the following:

> <em>(B) The first is a practice that the author proceeds to recommend against; the second is evidence that the author uses in the process of supporting that recommendation.</em>

In other words, the main conclusion of the argument is a recommendation against using the practice described in the first bolded portion, and the second bolded portion is part of the evidence used to support the main conclusion.

Now, to efficiently find the correct answer to the following question, locate the main conclusion of the argument made in the passage.

##  Example 12 

In recent years, so many people have been opting to attend college that the number of college graduates now exceeds the number of employment positions that require college educations.

In the argument given, the two portions in boldface play which of the following roles?

1. [ ] (A) The first is evidence used to support an intermediate conclusion of the argument; the second is that intermediate conclusion.
    > <br> <b>Correct Answer:</b> (D)
2. [ ] (B) The first is evidence used to support a conclusion opposed by the argument; the second is an intermediate conclusion drawn to support the argument’s main conclusion.
    > <br> <b>Correct Answer:</b> (D)
3. [ ] (C) The first is a judgment advanced in support of a position that the argument opposes; the second is the argument’s conclusion.
    > <br> <b>Correct Answer:</b> (D)
4. [x] (D) The first is a position that the argument opposes; the second is evidence used to support the argument’s conclusion.
5. [ ] (E) The first is a conclusion drawn to support the main conclusion of the argument; the second is that main conclusion.
    > <br> <b>Correct Answer:</b> (D)

##### Question Level: medium



To correctly answer this question, let’s start off by finding the main conclusion of the argument in the passage.


There are four main statements made in the passage:


- In recent years, so many people have been opting to attend college that the number of college graduates now exceeds the number of employment positions that require college educations.


- Some people, therefore, have made the case that<strong> much of the large investment that has been made in college education will not earn returns in terms of increased productivity.</strong>


- This assessment is incorrect


- <strong>in many positions that do not require college education, people who have knowledge gained through college education are generally more productive than are those without college education in the same positions.</strong>


The first and last of the statements are facts. The author does not conclude that the number of college graduates exceeds the number of employment positions that require college educations or that people who have knowledge gained through college education are more productive. Rather, the author presents these pieces of information as facts.


So, we are left with two more statements.


The second statement, the one about the case that “some people” have made, states a conclusion, but that conclusion is not the author’s conclusion; it’s the conclusion at which “some people” have arrived.


So, we are left with “This assessment is incorrect,” which is the main conclusion of the argument, as we can tell by noticing how it is supported by the last statement in the passage and noticing that it is the focus of all the information presented.


Now that we have located the main conclusion, choosing the correct answer will be rather straightforward.


> <b>(A) The first is evidence used to support an intermediate conclusion of the argument; the second is that intermediate conclusion.
</b>


Since there is no intermediate conclusion, the first boldfaced portion clearly does not support an intermediate conclusion. As discussed, the second boldfaced portion is a fact that supports the main conclusion, and it is not, itself, a conclusion.


> <b>(B) The first is evidence used to support a conclusion opposed by the argument; the second is an intermediate conclusion drawn to support the argument’s main conclusion.
</b>


 This choice is a little more tempting, because the first boldfaced portion does oppose the main conclusion of the argument, and the second boldfaced portion does support the main conclusion of the argument. However, since the first boldfaced portion does not support anything but rather, on its own, opposes the main conclusion, and, since the second boldfaced portion is stated as fact rather than supported as an intermediate conclusion, this choice does not accurately describe the two bolded sections.


> <b>(C) The first is a judgment advanced in support of a position that the argument opposes; the second is the argument’s conclusion.
</b>


The first boldfaced portion is itself a judgment, rather than support for such a judgment, and, as we discussed in our analysis in the introduction to this explanation, the second boldfaced portion is a statement of fact, rather than the argument’s conclusion.


> <span class="correct-answer"><i class="fal fa-check-circle"></i><span class="ms-1 fw-semibold correctness-text">CORRECT ANSWER</span></span><b>(D) The first is a position that the argument opposes; the second is evidence used to support the argument’s conclusion.
</b>


The first boldfaced portion is clearly a position that the argument opposes, since, as discussed in the introduction to this explanation, the main conclusion says of the first boldfaced portion, “This assessment is incorrect.” The second boldfaced portion is indeed a statement of fact that supports that main conclusion.


> <b>(E) The first is a conclusion drawn to support the main conclusion of the argument; the second is that main conclusion.
</b>


The first boldfaced portion does not support the main conclusion, but rather, is a position to which the conclusion is in opposition. Also, as discussed in the introduction to this explanation, the second boldfaced portion is not the main conclusion of the argument.


> ##### Correct Answer: (D)
