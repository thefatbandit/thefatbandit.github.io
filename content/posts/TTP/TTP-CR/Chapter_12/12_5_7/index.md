## Lesson Content
Another factor that can make sorting through the statements in a Boldface question and keeping everything straight challenging is the presence of multiple conclusions and the premises that support them.

> - Differentiating between premises and conclusions.

> - Determining which conclusions are supported by the arguments and which ones the arguments oppose.

> - Determining which premises support which conclusion.

Consider the following example:

> <em>There is a common perception that, because<strong> higher than normal water temperatures can kill off coral reefs</strong>, and because climate change related emissions are causing increases in ocean water temperatures, other than reducing climate change causing emissions, there is not much that can be done to save reefs. That perception is incorrect. <strong>Factors such as overfishing and pollution from farm fertilizer runoff result in much more damage to coral reefs than is being caused by high temperature water,</strong> and such factors can be addressed via local policy initiatives.</em>

<em>In the argument given, the two portions in boldface play which of the following roles?</em>

> <em>(<u>Trap Choice</u>) The first provides support for the main conclusion of the argument; the second also provides support for the main conclusion.</em>

> <em>(<u>Correct Answer</u>) The first states a consideration in support of a conclusion that the argument opposes; the second provides support for the argument’</em><em>s main conclusion.</em>

In this example, the wording of the two portions in boldface looks similar, and the information that they provide seems to support one conclusion, the conclusion that coral reefs are in trouble.

Of course, the truth is that, as the correct answer says, the two portions in boldface support two different conclusions: a conclusion that the main argument opposes and the argument’s main conclusion.

{{< alert type="warning" >}}
💡<b>MUST KNOW:</b>

The arguments in Boldface questions can contain multiple conclusions and premises to support the conclusions.
{{< /alert >}}

Of course, the way to keep everything straight is to identify the main conclusion of the argument and work from that reference point to determine what roles the other parts play.

Consider the following example.

##  Example 16 

A recent study showed a correlation between the quality of wine and the price of wine.

In the passage above, the portions in boldface play which of the following roles?

1. [ ] (A) The first is a conclusion that is opposed by the main conclusion of the passage; the second is the main conclusion.
    > <br> <b>Correct Answer:</b> (C)
2. [ ] (B) The first is a pattern that could be taken as support for a conclusion to which the main conclusion is in opposition; the second is the main conclusion.
    > <br> <b>Correct Answer:</b> (C)
3. [x] (C) The first could be taken as support for a conclusion that the main conclusion opposes; the second provides support for the main conclusion.
4. [ ] (D) The first is a specific case used to support a questionable assertion; the second is a more general case that follows from that assertion.
    > <br> <b>Correct Answer:</b> (C)
5. [ ] (E) The first is an objection that has been raised against a position defended in the argument; the second is that position.
    > <br> <b>Correct Answer:</b> (C)

##### Question Level: hard



The passage contains multiple conclusions and support for those conclusions. To get the question correct, let’s get them all straight. We’ll start with the main conclusion:


<u>Main Conclusion</u>: it would be wrong to conclude from the existence of this pattern that a winery will benefit financially from improving the quality of its wine.


The main conclusion contains within itself a conclusion that it opposes:


<u>Conclusion Opposed by the Main Conclusion</u>: a winery will increase its profitability by improving the quality of its wine.


The conclusion opposed by the main conclusion follows from the following conclusion about a pattern:


<strong>a winery’s improving the quality of its wine will enable that winery to increase the price that it receives per bottle</strong>


That conclusion is supported by the following:


A recent study showed a correlation between the quality of wine and the price of wine.


Finally, the main conclusion is supported by the last two sentences of the passage:


There are costs to improving wine quality, such as the extra expense of higher quality grapes, money spent on new technology, and salaries of top-notch winemakers, and, therefore, <strong>wineries that improve wine quality often see lower returns on investment than do wineries that produce wine of lower quality</strong>.


Now that we are clear about the functions of every part of the passage, let’s check the answer choices.


> <b>(A) The first is a conclusion that is opposed by the main conclusion of the passage; the second is the main conclusion.
</b>


The first bolded portion is not a conclusion opposed by the main conclusion. As discussed above, it provides support for a conclusion opposed by the main conclusion. Similarly, the second portion in boldface is not the main conclusion of the passage. It is support for the main conclusion.


> <b>(B) The first is a pattern that could be taken as support for a conclusion to which the main conclusion is in opposition; the second is the main conclusion.
</b>


The first half of this choice is correct, as the pattern mentioned in the first bolded portion could be taken as support for the conclusion opposed by the main conclusion of the passage. The second half is incorrect, however, as the second bolded portion is not the main conclusion but rather support for the main conclusion.


> <span class="correct-answer"><i class="fal fa-check-circle"></i><span class="ms-1 fw-semibold correctness-text">CORRECT ANSWER</span></span><b>(C) The first could be taken as support for a conclusion that the main conclusion opposes; the second provides support for the main conclusion.
</b>


This choice exactly matches the analysis discussed in the introduction to this explanation.


> <b>(D) The first is a specific case used to support a questionable assertion; the second is a more general case that follows from that assertion.
</b>


The first half of this choice is close to being correct, as the first bolded portion does provide some support for a conclusion to which the argument is in opposition, which conclusion could therefore be considered “questionable.” However, the second bolded portion clearly does not follow from the assertion that the argument opposes, which assertion is that a winery will increase its profitability by improving the quality of its wine.


> <b>(E) The first is an objection that has been raised against a position defended in the argument; the second is that position.
</b>


This first half of this choice is almost correct in a convoluted way, as the first bolded portion does support a conclusion that conflicts with the conclusion supported by the argument. However, the first bolded portion is not an objection, in other words a response, to the position that the argument defends. It is simply information that happens to support a different conclusion. The second part of this choice is clearly incorrect, as the second bolded portion is stated as fact and, thus, is not a position defended by the argument.


> ##### Correct Answer: (C)

Let’s now consider how the incorrect answer choices in Boldface questions can be constructed to make them tempting.
