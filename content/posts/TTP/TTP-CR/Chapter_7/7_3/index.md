## Lesson Content
Every Inference question on the GMAT begins with a passage or other stimulus, which is followed by a question that asks, in one way or another, which of the answer choices states something that must be true given what is said in the stimulus.

While Inference questions can be worded in various ways, GMAT Critical Reasoning questions tend to utilize predictable language, a fact that we can exploit by familiarizing ourselves with the types of questions we will see.

> <em>Which of the following can be properly inferred from the statements above?</em>

> <em>If the statements above are all true, which of the following can properly be inferred on the basis of them?</em>

> <em>If the statements above are true, which of the following must also be true?</em>

> <em>Which of the following must be true about … ?</em>

> <em>W</em><em>hich of the following can be properly concluded from the statements in the passage?</em>

> <em>If the statements above are true, which of the following conclusions can be properly drawn?</em>

> <em>Which of the following is information provided by the passage</em><em>?</em>

{{< alert type="warning" >}}
💡<b>MUST KNOW:</b>

Learn to recognize that a question stem is asking you for an answer that must be true given what is said in the passage.
{{< /alert >}}
