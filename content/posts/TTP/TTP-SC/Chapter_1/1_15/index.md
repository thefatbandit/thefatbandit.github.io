## Lesson Content
Imagine that, in a question, the original sentence incorrectly uses the plural verb “were” with the singular subject “the company.” Most of you, even at this point, know that a singular subject must be paired with a singular verb.

Now, what happens if choice (B) fixes this issue? In other words, what if the sentence produced by answer choice (B) correctly uses the singular noun “the company” with the singular verb “was.” In that case we choose answer (B) as correct, right?

Not necessarily.

So, as always, we must read each sentence version in its entirety.

{{< alert type="warning" >}}
💡<b>MUST KNOW:</b>

When we are answering a Sentence Correction question, the fact that a sentence produced by a later answer choice fixes an error found in the original sentence, or any earlier sentence, does not allow us, on that basis alone, to choose the new sentence as correct.
{{< /alert >}}

Consider the following example:

##  Sentence Correction Example 19  

<u>The twenty pages of the document contains a description of the room in which the possessions of the governor were</u> found when the archeologists unearthed the long-buried city.

1. [ ] (A) The twenty pages of the document contains a description of the room in which the possessions of the governor were
    > <br> <b>Correct Answer:</b> (D)
2. [ ] (B) A twenty-page document containing a description of the room in which the possessions of the governor were
    > <br> <b>Correct Answer:</b> (D)
3. [ ] (C) The twenty-page document contains a description of the room from which the possessions of the governor was
    > <br> <b>Correct Answer:</b> (D)
4. [x] (D) The twenty-page document contains a description of the room in which the possessions of the governor were
5. [ ] (E) A twenty-page document contains a description of the room from which the possessions of the governor was
    > <br> <b>Correct Answer:</b> (D)

##### Question Level: medium



The original version of the sentence in this question has a subject-verb agreement error. The plural subject “the twenty pages of the document” does not agree with the singular verb “contains.”


To arrive at the correct answer to the question, we have to be careful not to get too excited and choose the first choice we see that fixes this subject-verb agreement error, because the first choice that fixes this error, choice (C), has new errors.


> <b>(A)</b> <b>The twenty pages of the document contains a description of the room in which the possessions of the governor were
</b> found when the archeologists unearthed the long-buried city.



 The plural subject “the twenty pages” does not agree with the singular verb “contains,” as we can more easily see by crossing off what appears between “the twenty pages” and “contains.”


The twenty pages <s>of the document</s> contains


> <b>(B)</b> <b>A twenty-page document containing a description of the room in which the possessions of the governor were
</b> found when the archeologists unearthed the long-buried city.



 This version describes the document without ever completing the thought. Notice that it says, “a twenty-page document” and then continues by saying more about the document, “containing a description of the room …,” but leaves the reader wondering “A twenty-page document containing a description of the room does what?”


So, this version has no verb to tell us what the document does and, thus, is not a complete sentence.


> <b>(C)</b> <b>The twenty-page document contains a description of the room from which the possessions of the governor was
</b> found when the archeologists unearthed the long-buried city.



 This version fixes the subject-verb agreement error seen in (A) by changing “the twenty pages of the document” to “the twenty-page document.” The singular subject “the twenty-page document” agrees with the singular verb “contains.”


However, we have to be careful not to choose this choice just because it fixes an error that exists in a previous version, because this version has new errors.


For one thing, we can see that this version misuses “from” in “the room <em>from</em> which the possessions of the governor were found.” The possessions would have been found “in” a room, not “from” a room.


Also, the plural subject “possessions” does not agree with the singular verb “was found,” as we can more easily see by crossing off what appears between “possessions” and “was found.”


the possessions <s>of the governor</s> was found


> <span class="correct-answer"><i class="fal fa-check-circle"></i><span class="ms-1 fw-semibold correctness-text">CORRECT ANSWER</span></span><b>(D)</b> <b>The twenty-page document contains a description of the room in which the possessions of the governor were
</b> found when the archeologists unearthed the long-buried city.



 This version is grammatically correct from beginning to end and effectively conveys a meaning that makes sense.


> <b>(E)</b> <b>A twenty-page document contains a description of the room from which the possessions of the governor was
</b> found when the archeologists unearthed the long-buried city.



 This version fixes the lack of a verb seen in (B), by using “contains” rather than “containing.” By using “contains,” this version, unlike the version created via the use of (B), tells us what the document does: it “contains.”


However, we have to be careful not to choose this choice just because it fixes an error that exists in a previous version, because this version has new errors.


For one thing, we can see that this version misuses “from” in “the room <em>from</em> which the possessions of the governor were found.” The possessions would have been found “in” a room, not “from” a room.


Also, the plural subject “possessions” does not agree with the singular verb “was found,” as we can more easily see by crossing off what appears between “possessions” and “was found.”


the possessions <s>of the governor</s> was found


> ##### Correct Answer: (D)

Let’s now discuss not falling into choosing choice (E).
