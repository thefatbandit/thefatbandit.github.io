## Lesson Content
To consistently succeed in identifying issues in the sentence versions in Sentence Correction questions, we have to read the sentences in their entirety, or holistically.

{{< alert type="warning" >}}
💡<b>MUST KNOW:</b>

To consistently succeed in identifying issues in the sentence versions in Sentence Correction questions, we have to read the sentences in their entirety, or holistically. If we consider the sentences only one part at a time, we can easily fool ourselves into thinking an incorrect version is correct.
{{< /alert >}}

For example, we could confirm that the subject and main verb of a sentence work logically with each other, that the adjectives in the sentence work logically with what they modify, and that there are no issues with the pronouns in the sentence and decide that the sentence is correct.

So, you can see that, since Sentence Correction versions can have a variety of issues involving not just grammar but also structure and meaning conveyed, we have to consider the sentences as sentences, not just as collections of words and phrases that might have errors.

##  Sentence Correction Example 3  

<u>Ostriches are able to run fast for very long distances, but they are</u> able to escape predators such as hyenas that normally catch prey by exhausting the prey.

1. [ ] (A) Ostriches are able to run fast for very long distances, but they are
    > <br> <b>Correct Answer:</b> (E)
2. [ ] (B) Because ostriches are able to run fast for very long distances, it is
    > <br> <b>Correct Answer:</b> (E)
3. [ ] (C) To be able to run fast for very long distances, ostriches are
    > <br> <b>Correct Answer:</b> (E)
4. [ ] (D) An ostrich, able to run fast for very long distances, are
    > <br> <b>Correct Answer:</b> (E)
5. [x] (E) Since it is able to run fast for very long distances, the ostrich is

##### Question Level: medium



Some of the incorrect versions of the sentence in this question are clearly grammatically flawed. However, other incorrect versions are grammatically perfect and are incorrect only because they convey illogical meanings. So, checking only for grammar errors is not sufficient for arriving at the correct answer. Determining which choice is the correct answer requires reading each version in its entirety to see whether what it conveys makes sense.


> <b>(A)</b> <b>Ostriches are able to run fast for very long distances, but they are
</b> able to escape predators such as hyenas that normally catch prey by exhausting the prey.



This version is grammatically perfect. The plural subject “ostriches” agrees with the plural verb “are,” the plural pronoun “they” logically refers to the plural “ostriches,” and the rest of the sentence is free of grammar errors.


Notice, however, that this version conveys a nonsensical meaning.


The word “but” is used to communicate that two ideas connected by “but” contrast with each other, but in this case, the two ideas connected by “but” would never contrast with each other.


The two ideas connected by “but” are “Ostriches are able to run fast for very long distances” and “they are able to escape.” Of course, being able to run fast and being able to escape don’t contrast with each other. Instead, the fact that ostriches are able to run fast explains why they are able to escape. So, saying that ostriches are able to run fast “but” they are able to escape does not make sense.


> <b>(B)</b> <b>Because ostriches are able to run fast for very long distances, it is
</b> able to escape predators such as hyenas that normally catch prey by exhausting the prey.



This version is clearly grammatically flawed. It starts off mentioning the plural “ostriches,” but then says, “it is able to escape.” The singular pronoun “it” cannot refer to the plural “ostriches.” So, “it” has nothing to refer to.


> <b>(C)</b> <b>To be able to run fast for very long distances, ostriches are
</b> able to escape predators such as hyenas that normally catch prey by exhausting the prey.



This version is grammatically correct. However, by reading this version in its entirety, we can see that it conveys a meaning that does not make sense.


Since the word “to” indicates purpose, in the context of a sentence like this one, “to be able” means “for the purpose of being able.”


So, this version conveys the meaning “for the purpose of being able to run fast for very long distances, ostriches are able to escape.” Of course, that meaning does not make sense. Ostriches would not be able to escape in order to run fast for long distances. Rather, they would be able to run fast for long distances in order to be able to escape. So, this version conveys a meaning that is basically the reverse of what would make sense.


> <b>(D)</b> <b>An ostrich, able to run fast for very long distances, are
</b> able to escape predators such as hyenas that normally catch prey by exhausting the prey.



This version is clearly grammatically flawed because the singular subject, “an ostrich,” does not agree with the plural verb “are.” We can more easily see the error by crossing off the wording between the subject and the verb.


<em>An ostrich</em><s>, able to run fast for very long distances,</s> <em>are</em> able to escape


> <span class="correct-answer"><i class="fal fa-check-circle"></i><span class="ms-1 fw-semibold correctness-text">CORRECT ANSWER</span></span><b>(E)</b> <b>Since it is able to run fast for very long distances, the ostrich is
</b> able to escape predators such as hyenas that normally catch prey by exhausting the prey.



This version is grammatically perfect. The singular subjects “it” and “ostrich” agree with the singular verb “is,” the singular pronoun “it” logically refers to the singular “ostrich,” and the rest of the sentence is free of grammar errors.


Furthermore, the meaning conveyed by this version makes sense.


So, we can see that this choice is the correct answer.


> ##### Correct Answer: (E)
