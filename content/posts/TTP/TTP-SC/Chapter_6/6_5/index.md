## Lesson Content
Sentence Correction questions involving verb tense issues – even difficult ones – become easy once we realize that each of the sentence versions in these questions will include well-placed clues indicating exactly which tense should be used.

{{< alert type="warning" >}}
💡<b>MUST KNOW:</b>

Correctly answering verb, tense, mood questions is more about solving a puzzle than it is about being a grammarian.
{{< /alert >}}

Will we need some grammar knowledge? We will.

{{< alert type="warning" >}}
💡<b>MUST KNOW:</b>

In each of the sentence versions in Sentence Correction questions involving verb tense issues, the test-maker will include well-placed clues indicating exactly which tense must be used. To choose the version with the most logical use of tense, we must notice the clues provided.
{{< /alert >}}

Let’s discuss some specific strategies that we can employ.
