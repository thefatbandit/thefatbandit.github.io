## Lesson Content
When a sentence describes an event that has clearly ended in the past, the use of the present perfect is not correct.

{{< alert type="warning" >}}
💡<b>MUST KNOW:</b>

When a sentence describes an event that clearly ended in the past, the present perfect is not correct.
{{< /alert >}}

The reason is that, as we know, the present perfect tense describes an action that began in the past and continues to be happening in the present or continues to be affecting the present, so if the action ended in the past, using the present perfect is illogical.

{{< alert type="warning" >}}
💡<b>MUST KNOW:</b>

If an action clearly ended in the past, using the present perfect is illogical.
{{< /alert >}}

How do we put this knowledge into action on test day? When we encounter a sentence version that describes an event that clearly ended in the past but also uses the present perfect tense to describe that action, eliminate that sentence version!

<strong><u>Key Strategy</u></strong><strong>:</strong> When a sentence version that describes an event that clearly ended in the past but also uses the present perfect tense to describe that action, eliminate that sentence version!

Let’s practice with some example sentences:

> <u>Incorrect</u>: Last year, the board <u>has met</u> six times.

> <u>Correct</u>: Last year, the board <u>met</u> six times.

Sentence version one does not work because the present perfect tense describes an action began in the past and continues into the present, but the context clue “last year” makes it clear that the action of the board meeting has ended.

Sentence version two correctly uses the past tense verb “met,” which indicates to us that the board met in the past, but no longer does so.

Let’s try another:

> <u>Incorrect</u>: Before Marty and Scott won the championship, they <u>have visualized</u> success.

> <u>Correct</u>: Before Marty and Scott won the championship, they <u>visualized</u> success.

Sentence version one can’t work.

Sentence version two correctly uses the past tense verb “visualized” to express that Marty and Scott’s visualization ended in the past.
