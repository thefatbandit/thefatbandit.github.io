## Lesson Content
We already know that the coordinating conjunctions are “for,” “and,” “nor,” “but,” “or,” “yet,” and “so.” We use the mnemonic device FANBOYS to remember them.

{{< alert type="warning" >}}
💡<b>MUST KNOW:</b>

The coordinating conjunctions (FANBOYS): For, And, Nor, But, Or, Yet, So.
{{< /alert >}}

These FANBOYS conjunctions are always parallelism markers, and a structure built around one of the FANBOYS must be parallel.

In other words, any time we encounter one of these FANBOYS conjunctions, we must be vigilant in ensuring that parallelism is properly maintained in the sentence in which it’s found.

{{< alert type="warning" >}}
💡<b>MUST KNOW:</b>

The FANBOYS conjunctions are always parallelism triggers. A structure built around one of the FANBOYS must be parallel. Thus, whenever we encounter one of the FANBOYS, we must check to see whether the sentence uses proper parallelism.
{{< /alert >}}

We’ll soon see that the most common FANBOYS conjunction we’ll encounter is “and.” So, of course, whenever we see an “and” in a sentence, we must check to that see whether parallelism is maintained.

{{< alert type="warning" >}}
💡<b>MUST KNOW:</b>

The FANBOYS conjunction “and” is the most common type of parallelism marker that we’ll encounter. Whenever we see an “and” in a sentence, we must check to see whether proper parallelism is maintained.
{{< /alert >}}

Let’s now discuss an efficient method for determining whether elements built around FANBOYS are parallel.
