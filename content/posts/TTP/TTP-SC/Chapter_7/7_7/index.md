## Lesson Content
In Sentence Correction questions, we commonly encounter correlative conjunctions, the most important of which to learn are:

> not only … but also

> not only … but … as well

> not … but

> both … and

> either … or

> neither … nor

{{< alert type="warning" >}}
💡<b>MUST KNOW:</b>

The most important correlative conjunctions to learn are “not only … but also,” “not only … but … as well,” “not … but,” “both … and,” “either … or,” “neither … nor.”
{{< /alert >}}

The correlative conjunctions are always parallelism triggers because, when expressions employing correlative conjunctions appear in sentences, items that are part of these correlative constructions must be parallel.

{{< alert type="warning" >}}
💡<b>MUST KNOW:</b>

When an expression employing the correlative conjunction “not only … but also,” “not only … but … as well,” “not … but,” “both … and,” “either … or,” or “neither … nor” appears in a sentence, we must be on the lookout for errors in parallel structure, because items that are part of such correlative constructions must be parallel.
{{< /alert >}}

To make learning this material as efficient as possible, we’ll begin with a rigorous discussion of the correlative conjunction “not only … but also.”

Then, the good news will be that the other correlative conjunctions work similarly to how “not only … but also” works.
