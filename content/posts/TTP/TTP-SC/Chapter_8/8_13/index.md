## Lesson Content
We use “number” with countable nouns and “amount” with uncountable nouns.

For example:

> The number of <strong>species</strong> worldwide is declining.

Notice that “species” is a countable noun.

> The number of <strong>people</strong> at the beach was amazing.

Of course, “people” is a countable noun.

> The amount of <strong>enthusiasm</strong> on display at the candidate’s rally was shocking.

“Enthusiasm” can’t be counted.

> The amount of <strong>water</strong> in the beaker is not sufficient for this experiment.

“Water” can’t be counted.

{{< alert type="warning" >}}
💡<b>MUST KNOW:</b>

We use “number” with countable nouns, and we use “amount” with uncountable nouns.
{{< /alert >}}

Of course, the words number and amount are often used in sentences that express comparisons.

> <u>Incorrect</u>: The amount of seals that inhabit Cape Cod is now greater than it was for decades.

The sentence incorrectly uses “amount” with the countable “seals.” “Number” must be used with countable nouns, as in:

> The <strong>number of seals</strong> that inhabit Cape Cod is now greater than it was for decades.

Now, whenever these words are used, in addition to considering whether “number” or “amount” is used correctly, we must also check to see whether comparisons made are logical.

> <u>Incorrect</u>: Currently, the number of cars on the road using electric power is far greater than five years ago.

Notice that this sentence illogically attempts to compare “the number of cars on the road using electric power” with “five years ago.” A corrected version is:

> Currently, the number of cars on the road using electric power is far greater than it was five years ago.

Of course, the following is also incorrect:

> <u>Incorrect</u>: Currently, the number of cars on the road using electric power is far greater than they were five years ago.

Notice that this sentence illogically attempts to compare “<strong>the number</strong> of cars on the road today using electric power” with “<strong>they</strong> were five years ago.” What does the “they” represent? Certainly not “the number.”

##  Sentence Correction Example 19  

Because, in rural areas, there is less ambient light than there is in urban areas, the <u>amount of stars that one can see with the naked eye is greater in rural areas than</u> in urban areas.

1. [ ] (A) amount of stars that one can see with the naked eye is greater in rural areas than
    > <br> <b>Correct Answer:</b> (D)
2. [ ] (B) number of stars that one can see with the naked eye in rural areas is greater than it is
    > <br> <b>Correct Answer:</b> (D)
3. [ ] (C) amounts of stars that one can see with the naked eye are greater in rural areas than they are
    > <br> <b>Correct Answer:</b> (D)
4. [x] (D) number of stars that one can see with the naked eye is greater in rural areas than it is
5. [ ] (E) amount of stars that one can see with the naked eye is greater in rural areas than that
    > <br> <b>Correct Answer:</b> (D)

##### Question Level: medium



> <b>(A)</b> Because, in rural areas, there is less ambient light than there is in urban areas, the <b>amount of stars that one can see with the naked eye is greater in rural areas than
</b> in urban areas.



This version is correct except in that it incorrectly uses “amount” with the countable “stars.”


> <b>(B)</b> Because, in rural areas, there is less ambient light than there is in urban areas, the <b>number of stars that one can see with the naked eye in rural areas is greater than it is
</b> in urban areas.



Since “it” would logically refer to “the number of stars that one can see with the naked eye in rural areas,” this version expresses the illogical idea that the number of stars that one can see with the naked eye in rural areas is greater than the number of stars that one can see with the naked eye in rural areas in urban areas, as we can clearly see by substituting “the number of stars that one can see with the naked eye in rural areas” for “it” in the sentence.


… <strong>the number of stars that one can see with the naked eye in rural areas</strong> is greater than <strong>the number of stars that one can see with the naked eye in rural areas</strong> is in urban areas.


> <b>(C)</b> Because, in rural areas, there is less ambient light than there is in urban areas, the <b>amounts of stars that one can see with the naked eye are greater in rural areas than they are
</b> in urban areas.



Even though “amounts” is plural, it still is not correct to use “amounts” with the countable stars.


> <span class="correct-answer"><i class="fal fa-check-circle"></i><span class="ms-1 fw-semibold correctness-text">CORRECT ANSWER</span></span><b>(D)</b> Because, in rural areas, there is less ambient light than there is in urban areas, the <b>number of stars that one can see with the naked eye is greater in rural areas than it is
</b> in urban areas.



This version correctly uses “number” with the countable “stars.”


Also, this version expresses the logical idea that a quantity, the number of stars that one can see with the naked eye, is greater in rural areas than “it,” meaning that quantity, is in urban areas.


> <b>(E)</b> Because, in rural areas, there is less ambient light than there is in urban areas, the <b>amount of stars that one can see with the naked eye is greater in rural areas than that
</b> in urban areas.



This version incorrectly uses “amount” with the countable “stars.”


Also, there is no logical referent for “that.” The sentence does not mention something “in rural areas” that could be logically compared with “that in urban areas.” Rather, it mentions “the amount of stars that can be seen with the naked eye,” and then says that this quantity is greater in rural areas than “that in urban areas.” That what? To more clearly see the issue, compare this choice with choice (D).


> ##### Correct Answer: (D)

##  Sentence Correction Example 20  

<u>To decorate the tables at the event, the caterers used an array of decoratively carved fruits, like kiwis, grapefruit, and melons, along with a large amount of chocolates in shapes such as stars, hearts, and flowers.</u>

1. [ ] (A) To decorate the tables at the event, the caterers used an array of decoratively carved fruits, like kiwis, grapefruit, and melons, along with a large amount of chocolates in shapes such as stars, hearts, and flowers.
    > <br> <b>Correct Answer:</b> (C)
2. [ ] (B) Decorating the tables at the event, an array of decoratively carved fruits, including kiwis, grapefruit, and melons, along with a large number of chocolates in shapes like stars, hearts, and flowers, used by the caterers.
    > <br> <b>Correct Answer:</b> (C)
3. [x] (C) Used by the caterers to decorate the tables at the event was an array of decoratively carved fruits, such as kiwis, grapefruit, and melons, along with a large amount of chocolate in shapes such as stars, hearts, and flowers.
4. [ ] (D) To decorate the tables at the event, an array of decoratively carved fruit, including kiwis, grapefruit, and melons, along with a large number of chocolates in shapes such as stars, hearts, and flowers, were used by the caterers.
    > <br> <b>Correct Answer:</b> (C)
5. [ ] (E) By using an array of decoratively carved fruits, like kiwis, grapefruit, and melons, along with a large amount of chocolate in shapes such as stars, hearts, and flowers, the tables at the event were decorated by the caterers.
    > <br> <b>Correct Answer:</b> (C)

{{< /quizdown >}}

   <h5><details><summary><b><u>View Solution</u></b></summary>
   {{< alert type="secondary" >}}

##### Question Level: hard



> <b>(A)</b> <b>To decorate the tables at the event, the caterers used an array of decoratively carved fruits, like kiwis, grapefruit, and melons, along with a large amount of chocolates in shapes such as stars, hearts, and flowers.
</b>


This version incorrectly uses “like” in introducing a list of examples of fruits.


Also, this version incorrectly uses “amount” with the countable “chocolates.”


> <b>(B)</b> <b>Decorating the tables at the event, an array of decoratively carved fruits, including kiwis, grapefruit, and melons, along with a large number of chocolates in shapes like stars, hearts, and flowers, used by the caterers.
</b>


This version incorrectly uses “like” in introducing a list of examples of shapes.


Also, this version includes no main verb, only nouns and modifiers, and, therefore, is not a complete sentence.


> <span class="correct-answer"><i class="fal fa-check-circle"></i><span class="ms-1 fw-semibold correctness-text">CORRECT ANSWER</span></span><b>(C)</b> <b>Used by the caterers to decorate the tables at the event was an array of decoratively carved fruits, such as kiwis, grapefruit, and melons, along with a large amount of chocolate in shapes such as stars, hearts, and flowers.
</b>


This version may “sound funny” because it is an inverted sentence in which the subject, “an array,” follows the main verb, “was.” All the same, this version is correct, correctly using “such as” in introducing lists of examples of fruits and shapes and correctly using “amount” with the uncountable “chocolate.”


Notice that, since “along with a large amount of chocolate …” is an additive phrase, the subject of the sentence can be correctly presumed to be the singular “array,” which agrees with the singular “was.”


> <b>(D)</b> <b>To decorate the tables at the event, an array of decoratively carved fruit, including kiwis, grapefruit, and melons, along with a large number of chocolates in shapes such as stars, hearts, and flowers, were used by the caterers.
</b>


This version suggests the illogical idea that an array of fruits had the intention of decorating the tables at the event and, therefore, was used by the caterers.


Also, the subject of the sentence is the singular “an array of decoratively carved fruit, which does not agree with the plural “were.” “Along with a large number of chocolates … is an additive phrase, and, therefore, its presence does not render the singular subject plural.


> <b>(E)</b> <b>By using an array of decoratively carved fruits, like kiwis, grapefruit, and melons, along with a large amount of chocolate in shapes such as stars, hearts, and flowers, the tables at the event were decorated by the caterers.
</b>


Since the opening modifier, “by using an array …” must modify the subject of the clause that follows, which is “the tables,” this version expresses the nonsensical meaning that the tables at the event used an array of fruit and a large amount of chocolate in order to be decorated by the caterers.


Also, this version incorrectly uses “like” in introducing a list of examples of fruits.


> ##### Correct Answer: (C)
