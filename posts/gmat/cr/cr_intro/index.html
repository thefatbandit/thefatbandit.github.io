CR-1 General Tips & Question Structure

  • MBA
  • GMAT
  • CR
  • Read all answer choices. Often the test-maker puts an attractive wrong choice before the right one
  • Assumptions:
    • Never assume anything blatantly
    • Assumptions under common sense are alright.
    • E.g.
      • The sky is sometimes cloudy
      • TV shows are transmitted as signals to homes

Structure of a CR Question

🚨 Read the parts in same order as given in the question

🚨 Primary Objectives of CR 🚨

  1. Determine whether the stimulus contains an argument or only a set of facts
  2. If it contains an argument identify the conclusion of the argument.
    • If it contains a fact set, examine each fact
  3. Determine if the argument is strong/weak
  4. Do not generalize! Read closely and know precisely what the author said
  5. Read & Identify the Question Stem type
  6. Pre-phase Answers: Mentally formulate your answer to the question stem after reading it
  7. Always read all 5 answer choices
  8. Separate the answer choices into Contenders/Losers. Decide the answer amongst the contenders
  9. If all the answer choices appear to be Losers, return to the stimulus and re-evaluate the argument
